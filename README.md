Gilded Forest is a terminal color scheme for programming and note-taking.

## Dark
| Palette      | Hex       | Hex-Bright     |
| ------------ | --------- | -------------- | 
| Background   | `#0e1406` |    `#0e1406`   | 
| Current Line | `#44475a` |    `#44475a`   | 
| Selection    | `#28296e` |    `#28296e`   | 
| Foreground   | `#f0d388` |    `#f0d388`   | 
| Black        | `#000000` |    `#625656`   | 
| Red          | `#dc738b` |    `#a86864`   | 
| Green        | `#b8f9b5` |    `#a7cc6f`   | 
| Yellow       | `#e9b677` |    `#dad05d`   | 
| Blue         | `#6d8aef` |    `#67aaba`   | 
| Magenta      | `#a67df7` |    `#6b2368`   | 
| Cyan         | `#83bcf3` |    `#6baaba`   | 
| White        | `#fcfdf9` |    `#fefefe`   |
